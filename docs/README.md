# Documentation

Use this folder for project-facing documents.

Suggested files:

- `requirements.md`: goals, users, scope, and acceptance criteria
- `design.md`: architecture, data model, and important tradeoffs
- `game-design.md`: implemented gameplay rules and mode behavior in English
- `game-design.zh-CN.md`: implemented gameplay rules and mode behavior in Chinese
- `win-rate-report.zh-CN.md`: current single-player win-rate analysis report in Chinese
- `api.md`: API contracts or integration notes
- `decisions.md`: durable decisions and why they were made
